# ASCII Video Site

Statik bir web sayfasinda `assets/Video_Editing_Request_Fulfilled.mp4` dosyasini canli ASCII art olarak oynatir.

## Calistirma

En saglikli yol lokal server ile acmaktir:

```powershell
cd C:\Users\onder\Documents\GitHub\ascii-video-site
node server.js
```

Sonra tarayicida `http://localhost:5500` adresini ac.

`file://` ile dogrudan acildiginda bazi tarayicilar video veya canvas erisimini engelleyebilir.
